Class: CaptionsPlugin

CaptionsPlugin

Adds support for in game captions. Renders captions over the game within a dom element

The dom element is styled following the below defaults

 
    fontSize: 24,
    fontFamily: 'sans-serif',
    color: 'white',
    background: 'black',
    paddingTop: 5,
    paddingBottom: 5,
    paddingLeft: 10,
    paddingRight: 10,
    bottom: 10,
    position: 'absolute',    
    left : 50%;
    transform : translateX(-50%)

This is a mandatory jgg component, and so will be available to any game extending AppBase.
see Component Manager


// access via scene
scene.captions

// access via game
game.captions

new CaptionsPlugin()

Members

background

set caption background colour

Default Value:
  • black

colour

set caption font colour

Default Value:
  • white

durationMaximum

set the maximum duration that a caption will display for ( disabled by default )

Default Value:
  • false

durationMinimum

set the minimum duration that a caption will display for ( if no duration is supplied )

Default Value:
  • 1500

durationPerChar

set the duration for each character in the caption.
If no duration is supplied the duration is calculated as ( value x caption.length )

enabled

Enable / Disable captions

Default Value:
  • false

fontFamily

set caption font family

Default Value:
  • sans-serif

fontSize

set caption font size

Default Value:
  • 24

padding

set caption background padding ( all sides )

paddingHorizontal

set caption background left and right padding

Default Value:
  • 10

paddingVertical

set caption background top and bottom padding

Default Value:
  • 5

scenePadding

set caption distance from screen edge

Default Value:
  • 10

style

set captions style, formatted in a css valid object

Default Value:
  • see example

    }

Methods

hide()

hide any active caption and clears the queue

show(copy [, duration])

Show a caption for a defined amount of time. By default the duration will be calculated according to caption length

Parameters:
Name Type Argument Description
copy string | Array

caption or captions to be displayed
duration number <optional>

duration to show caption for
Example

Example of the different way in which you can trigger captions

// show a single caption using default duration
    scene.captions.show('caption string');

    // show an array of captions using default duration
    scene.captions.show(['caption one', 'caption two'])

    // show an array of captions with defined duration for each caption
    scene.captions.show([{copy : 'caption one', duration : 1000}, {copy : 'caption two', duration : 2500} ])

    // mix and match
    scene.captions.show([{copy : 'caption one', duration : 1000}, 'caption two',  {copy : 'caption three', duration : 2500} ])
