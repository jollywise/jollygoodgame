Class: SceneControllerBase

SceneControllerBase

new SceneControllerBase(sceneManager)

A scene controller for managing scenes used in game.
This controller provides easy ability to add scenes, as well as control which scenes are active and trigger overlays.
Parameters:
Name Type Description
sceneManager Phaser.Scenes.SceneManager The game scene manager

Methods

addSceneMap(sceneMap)

Add a scene map to the controller.
The controller will parse each scene in the map and add them to phasers scene manager.
Parameters:
Name Type Description
sceneMap Object A scene map to add to the controller key/scene dictionary
Example
// state = imported Phaser.Scenes.Scene class

const SCENE_MAP = [
 { key: 'Welcome', state: Welcome },
 { key: 'Complete', state: Complete },
 { key: 'HowToPlay', state: HowToPlay },
 { key: 'GamePlay', state: GamePlay },
];

sceneController.addSceneMap(SCENE_MAP);

showOverlayScene(sceneKey, data)

Show a scene as an overlay. This will pause the current active scene and display the requested scene over the top
Parameters:
Name Type Description
sceneKey string The key of the scene to be obverlayed, should match a key in the Scene Map.
data Object Data that will be passed through to the scenes init method when starting.
Also accepts a callback for onOverlayCLosed which will be called when the overlay is closed of provided.

startScene(scene, data)

Start a scene. Usually you would use switchScene instead how-ever this can be called directly to start a particular scene.
Calling this directly will not stop and current active scenes.
Parameters:
Name Type Description
scene string The key of the scene to be started, should match a key in the Scene Map.
data Object Data that will be passed through to the scenes init method when starting.

stopScene(scene)

Will stop the requested scene from running, it is not destroyed and can be restarted again if desired.
Parameters:
Name Type Description
scene string The key of the scene to be stopped, should match a key in the Scene Map.

switchScene(sceneKey, data)

Switch game scenes, this will stop the current active scene and start the requested scene.
The new scene will automatically be brought to the top of the scene display list.
Parameters:
Name Type Description
sceneKey string The key of the scene to start, should match a key provided in the Scene Map.
data Object Data that will be passed through to the scenes init method whenn starting.
