Namespace: CaptionsPlugin

CaptionsPlugin

Adds support for in game captions. Renders captions over the game within a dom element

The dom element is styled following the below defaults

 
    fontSize: 24,
    fontFamily: 'sans-serif',
    color: 'white',
    background: 'black',
    paddingTop: 5,
    paddingBottom: 5,
    paddingLeft: 10,
    paddingRight: 10,
    bottom: 10,
    position: 'absolute',    
    left : 50%;
    transform : translateX(-50%)

This is a mandatory jgg component, and so will be available to any game extending AppBase.
see Component Manager


// access via scene
scene.captions

// access via game
game.captions
