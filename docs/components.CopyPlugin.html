Class: CopyPlugin

CopyPlugin

Manages in-game strings. Register a copy document ( xml or json ), and then you can use this for lookups within the game

This is a mandatory jgg component, and so will be available to any game extending AppBase.
see Component Manager


// access via scene
scene.copy

// access via game
game.copy

new CopyPlugin()
